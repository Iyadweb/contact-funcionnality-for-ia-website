This project is a fully functional backend service for handling contact form submissions. It receives user input, validates and sanitizes the data, generates unique IDs for each submission, manages errors gracefully, and stores the results in a local JSON file.
The goal of this project is to demonstrate a clean backend architecture, proper input handling, and a simple but reliable workflow for managing form submissions.

REST endpoint for handling contact form POST requests

Structured validation and data sanitization

Automatic submission ID + timestamp generation

JSON-based storage system

Clear error handling and success responses

Console logging for submission tracking

Lightweight, readable, and maintainable structure

Tech Stack: 

Node.js

Express.js

JavaScript

JSON file storage
